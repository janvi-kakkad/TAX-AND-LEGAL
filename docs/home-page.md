# Home Page — Legal Analytics Dashboard

![Home](../designs/wireframes/Home.png)

## Overview

The Home Page provides a macro-level analytical view of the legal case database. It helps users understand trends, patterns, and statistics across multiple cases.

## Key Features

### 1. KPI Metrics
- Total Cases
- Petitioner Win Rate
- Judicial Velocity
- Average Case Duration

These metrics provide a quick snapshot of the legal dataset.

### 2. Data Visualizations

- Case Categories (Pie Chart)
- Case Trends by Year (Line Graph)
- Top 10 Most Cited Sections (Bar Chart)
- Case Outcome Distribution (Pie Chart)
- Top 10 Most Cited Precedents (Bar Chart)

These charts help in identifying trends and patterns in legal data.

### 3. Filters

Users can filter cases using:

- Court Type
- Case Type
- Outcome
- Start Year
- End Year


## Purpose

The dashboard enables legal professionals to perform high-level analysis without manually reviewing multiple case documents.
